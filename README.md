# sOp-timizadores
Investigamos la relación entre síntomas del Síndrome de Ovario Poliquístico (acné, caída de cabello, aumento de peso) y hábitos de vida (comida rápida, ejercicio). Además, analizamos su impacto en hormonas clave (AMH, LH) para identificar los factores de riesgo más importantes del SOP.
